中信国际电讯CPC获得VMware Cloud Verified 标志
香港，2020年8月27日 — 中信国际电讯集团有限公司（香港交易所股份代号：1883）全资拥有的中信国际电讯（信息技术）有限公司（「中信国际电讯CPC」）今天宣布获得VMware Cloud Verified 标志。
获得Cloud Verified标志体现了该VMware云服务供应商可以通过一整套软件定义服务，包括网络、存储及运算，为客户提供一致的基础架构和运营能力。客户在使用传统或容器化(containerized)的企业应用时，均可以享有一致、高性能及互联互通的优势，并确保有关服务是基于更先进的VMware云技术所提供。
中信国际电讯CPC产品开发及管理部高级副总裁蓝泰来表示：「获得VMware Cloud Verified标志是中信国际电讯CPC一个新的里程碑，象征着我们能为客户提供符合VMware先进云技术标准的服务和解决方案。此外，过去多年与VMware的紧密合作为我们的网络及云端策略带来极大优势，不但丰富了云网资源及服务能力，更为我们持续扩展业务的路上注入更多可能。」蓝泰来强调：「我们看到市场对混合云及多云方案的需求正在迅速增长，而对虚拟网络的需求也十分殷切；获得VMware Cloud Verified标志正可表明我们的SmartCLOUD™ 云计算方案可供给客户VMware云端基础架构的优势价值。我们不仅为客户提供可靠的基础设施即服务(IaaS)，还能够提供灵活的虚拟网络（NSX）、虚拟存储（vSAN）和软件定义广域网（SD-WAN）解决方案，满足客户数字化转型的需要。此外，通过强强连手，将VMware的产品与我们的一站式信息通信技术(ICT)解决方案完美融合，加上我们的专业服务及多年的托管服务经验，定能为客户带来独特的价值及体验。」
VMware亚太及日本区云供应商计划负责人Vijoo Chacko表示：「得到VMware Cloud Verified的合作伙伴均能够充份掌握VMware云技术的优点，并能够提供跨云平台运作的互联互通服务，为其客户带来莫大裨益。VMware的云服务供应商能为客户提供获得认证的云服务，其效率、灵活性和可靠性毋庸置疑。以中信国际电讯CPC为例，他们为其客户提供了一个简单而灵活的途径通往云端大道，尽握云端优势。我们将会对中信国际电讯CPC继续给予全力支持。」
VMware在全球拥有4,300家VMware云供应商，他们均利用VMware一致的云基础架构来提供广泛的服务，并凭借他们不同地区和行业的专业知识，在全球120多个国家及地区帮助客户满足复杂的法规要求。

中信国际电讯CPC简介
中信国际电讯（信息技术）有限公司（“中信国际电讯CPC”）是中信国际电讯集团有限公司的全资附属公司（香港交易所股份代号：1883），公司一直矢志透过先进技术及旗舰解决方案，包括TrueCONNECT™专用网络服务、TrustCSI™信息安全解决方案、DataHOUSE™全球统一云数据中心解决方案及SmartCLOUD™云计算解决方案，为全球跨国企业提供可高度扩展的客制化方案，以满足他们特定的ICT需求。
中信国际电讯CPC的服务遍及全球，不论在东、西方地区均有在地优势，成为领先的“服务在地，连接全球”的信息技术方案伙伴，真正实践了“创新‧不断”的服务理念。作为数码世界赋能者，公司致力推动数码化，在主要市场积极发展信息技术，将人工智能、增强现实、大数据，物联网和其他尖端新兴技术的强大潜力转化为企业客户的实际价值，持续为他们的业务发展带来更高的生产力、灵活性及成本效益，最终达致全球数码化。
